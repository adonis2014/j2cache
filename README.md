# j2cache
在项目中经常会用到内存作为缓存，以提供高性能。本项目主要是针对缓存提供一种可扩展的思路，并且统一缓存的访问接口。项目的目的主要是key-value类型的内存数据缓存存储。

## 为此项目中实现了三种存储方案：
- jvm虚拟机内存
- redis
- iginte
